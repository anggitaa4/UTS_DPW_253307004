**Deskripsi Projek**
- Website ini dibuat untuk menampilkan informasi tentang Penerimaan Peserta Didik Baru (PPDB), yang dimana isinya terdapat beberapa halaman yaitu halaman beranda yang berisi informasi umum dan jadwal PPDB, halaman alur yang menjelaskan langkah-langkah pendaftaran, serta halaman form yang digunakan untuk mengisi data seperti nama, NIK, dan memilih jalur pendaftaran.
- Website ini dibuat menggunakan HTML, CSS, dan JavaScript. HTML digunakan untuk menyusun struktur halaman, CSS digunakan untuk mengatur tampilan agar lebih rapi dan menarik, sedangkan JavaScript digunakan untuk menambahkan interaksi seperti mengambil data dari form dan menampilkan notifikasi.
- File CSS di pisahkan dari file HTML yang menjadi file style.css dimana file style.css itu harus selalu dihubungkan dengan file HTML menggunakan tag <link>, contohnya <link rel="stylesheet" href="asset/css/style.css">, gunanya agar tampilan terlihat menarik, bagus, dan terstruktur seperti penambahan warna gradasi, mengatur layout , navigasi dan lainnya.
